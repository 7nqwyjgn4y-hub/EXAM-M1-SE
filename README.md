# EXAM-M1-SE

Pour pouvoir utiliser notre outil, il faut télécharger les données RGPD de Instagram dans le format JSON.
Une fois ceci effectué, il faut extraire l'archive obtenue par mail et déposer le fichier dans un fichier vierge.
La prochaine etape est de créer un fichier en R markdown et de copier coller l'integralité du code ci dessous.
Sauvegarder le code et placer le fichier de code a coté de l'extraction instagram au sein du meme fichier.
Maitenant vous pouvez run le code en markdown et générer un fichier plus lisible en HTML (Knit) si vous le souhaitez.
